# Проект: Путешествие по России
Это учебная работа, представляющая собой одностраничный сайт. Реализован с помощью HTML и CSS.
## Технологии
Здесь реализованы:
* адаптивная вёрстка;
* флексбоксы;
* гриды;
* плавность переходов;
* подключение нестандартного шрифта Inter;
* организация файловой структуры Nested
